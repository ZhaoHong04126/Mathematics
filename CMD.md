## 終端機指令

### 轉換資料夾
在終端機下執行：

```bash
cd D:\UniversityMathematics
```

## 🚀 常用指令

### 啟動開發伺服器
執行以下指令啟動本機伺服器：
```bash
npm run dev
```

### 安裝相依套件
在專案根目錄下執行：
```bash
npm install
```

### 編譯生產版本
若需要編譯並壓縮專案以供部署：
```bash
npm run build
```
編譯完成後的靜態檔案將輸出於 `dist/` 資料夾中。
